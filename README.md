# MyPortfolio Backend API

## Overview
This repository contains the **ASP.NET Core backend API** for the **MyPortfolioApp** project.  
It provides RESTful APIs to manage **profile information, certificates, and projects** for multiple users in a shared database.  

This backend is designed to work with the Angular frontend of the portfolio application.

---

## Tech Stack
- **Framework:** ASP.NET Core 8 Web API  
- **Database:** SQLite (local file-based)  
- **ORM:** Entity Framework Core  
- **Version Control:** Git / GitHub  
- **Other Tools:** Swagger for API documentation
